# kj_namechanger
Name Changer for RSG Framework

Desc:
- Change players name without going through the tedious of making them log out and log in again
- Made with ox_lib
- Job-specified access
- Spawn / despawn ped by distancing
- Drawtext / Drawmarker by distance
- Select player by player ID
- Verify if selected player ID exist from serverside

Dependencies:
- rsg-core
- ox_lib
- ox_target
